Echo Fingerprint¶

Echo any fingerprint to stdout

  • Docker
  • Bitbucket
  • Github 
docker run \
    --env MERKELY_COMMAND=echo_fingerprint \
    --env MERKELY_FINGERPRINT="${MERKELY_FINGERPRINT}" \
    --rm \
    merkely/change:latest

ENV_VAR_NAME

Required?

Notes

MERKELY_COMMAND

yes

The Merkely command to execute. This page documents a value of echo_fingerprint

MERKELY_FINGERPRINT

yes

The artifact’s Fingerprint

 
    - step: &echo_any_fingerprint_to_stdout
        name: Echo any fingerprint to stdout
        services: [ docker ]
        script:
          - *export_merkely_fingerprint_env_vars
          - pipe: docker://merkely/change:latest
            variables:
              MERKELY_COMMAND: echo_fingerprint
              MERKELY_FINGERPRINT: ${MERKELY_FINGERPRINT}
